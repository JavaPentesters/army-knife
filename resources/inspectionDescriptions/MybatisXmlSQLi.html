Geek 1004: Mybatis XML SQL注入漏洞

攻击者可利用此漏洞，恶意构造SQL语句，导致数据库信息泄漏或被篡改。


错误实践:

    <select id="getUsers" resultMap="User">
        select * from table
        where id in ($ids)
        and status = ${status}
    </select>

最佳实践:

    <select id="getUsers" resultMap="User">
        select * from table
        where id in
        <foreach collection="ids" item="id" open="(" separator="," close=")">
            #{id}
        </foreach>
        and status = #{status}
    </select>
