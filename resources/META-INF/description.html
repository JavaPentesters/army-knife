A common command for Java based applications

Army knife is a userful tool that helps developers to find commonly used command of different middleware so that developers needn't to remember commands that are hard to remember!

  • The plugin supports many commands including JVM, Arthas, Linux, Redis, Dubbo, Mysql etc.
  • JVM: jmap, jstat, jstack
  • Arthas: jad, watch, refine, tt etc.
  • Linux: ss, lsof, top, log etc.
  • Redis: client etc.
  • Mysql: deadlock check etc.
